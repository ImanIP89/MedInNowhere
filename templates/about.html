📖 درباره MedInNowhere

🩺 پروژه چیست؟

MedInNowhere – پزشک در ناکجاآباد – یک دستیار هوشمند سلامت دیجیتال است که با استفاده از پردازش زبان طبیعی فارسی (NLP) و هوش مصنوعی، به تشخیص اولیه‌ی بیماری‌ها بر اساس علائم کاربر کمک می‌کند.

این پروژه به‌گونه‌ای طراحی شده که روی سخت‌افزارهای ضعیف (مانند Raspberry Pi) نیز اجرا شود و قابلیت استفاده‌ی چندکاربره در شبکه‌ی محلی را دارد.

✨ ویژگی‌های کلیدی

تشخیص هوشمند فارسی ورودی صوتی تشخیص گام‌به‌گام امتیازدهی وزنی علائم تاریخچه تشخیص‌ها ارجاع به پزشک یادآور دارو بدن‌نما (Body Map) ترجمه چندزبانه ترمینال CMD بارگذاری گروهی بیماری‌ها احراز هویت و نقش‌ها امنیت بالا (CSRF, Rate Limit) PWA (نصب روی موبایل)

⚙️ تکنولوژی‌ها

Python + Flask SQLite HTML5 + CSS3 + JavaScript Web Speech API DeepSeek API Flask-Limiter Werkzeug Security Service Worker

🎯 هدف پروژه

فراهم‌سازی یک ابزار کمک‌تشخیص پزشکی رایگان، فارسی‌زبان، آفلاین و قابل اجرا روی سخت‌افزارهای ارزان‌قیمت برای استفاده در مناطق محروم، درمانگاه‌های کوچک و حتی منازل.

📂 متن‌باز (Open Source)

این پروژه تحت مجوز MIT منتشر شده و کد منبع آن روی GitHub در دسترس است.
