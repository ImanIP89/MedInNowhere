# 🩺 MedInNowhere — پزشک در ناکجاآباد

**Medic in the Middle of Nowhere — AI-Powered Health Assistant**

---

### 🇮🇷 فارسی

**MedInNowhere** (ادامه‌دهنده راه MediSync) یک پلتفرم سلامت دیجیتال متن‌باز است که از **پردازش زبان فارسی (NLP)**، **تشخیص گام‌به‌گام**، **تحلیل تصویر ارتوپدی** و **چت زنده** برای کمک به تشخیص اولیه بیماری‌ها استفاده می‌کند.

#### ✨ ویژگی‌های نسخه ۳.۰
- 🧠 تشخیص هوشمند از جملات فارسی
- 🦴 تحلیل تصویر رادیولوژی با هوش مصنوعی
- 💬 چت زنده پزشک و بیمار
- 🗺️ بدن‌نما برای انتخاب محل درد
- ⏰ یادآور دارو با اعلان مرورگر
- 📋 ارجاع به پزشک و یادداشت‌گذاری
- 📊 امتیازدهی وزنی علائم
- 📱 PWA قابل نصب روی موبایل
- 🔒 امنیت بالا (CSRF, Rate Limiting)
- 🌐 ترجمه چندزبانه

### 🇬🇧 English

**MedInNowhere** (successor of MediSync) is an open-source digital health platform that leverages **Persian NLP**, **guided diagnosis**, **orthopedic X-ray analysis**, and **real-time chat** to assist in preliminary disease detection.

#### ✨ v3.0 Features
- 🧠 Smart Persian NLP diagnosis
- 🦴 AI-powered X-ray image analysis
- 💬 Real-time doctor-patient chat
- 🗺️ Body map for pain location selection
- ⏰ Medication reminder with browser notifications
- 📋 Doctor referrals & clinical notes
- 📊 Weighted symptom scoring
- 📱 Installable PWA
- 🔒 CSRF protection & rate limiting
- 🌐 Multi-language translation

### 🇰🇷 한국어

**MedInNowhere** (MediSync의 후속작)는 **페르시아어 NLP**, **단계별 진단**, **정형외과 X-ray 분석**, **실시간 채팅**을 통해 초기 질병 진단을 돕는 오픈소스 디지털 헬스 플랫폼입니다.

#### ✨ v3.0 기능
- 🧠 페르시아어 스마트 진단
- 🦴 AI 기반 X-ray 이미지 분석
- 💬 의사-환자 실시간 채팅
- 🗺️ 신체 부위 통증 지도
- ⏰ 브라우저 알림 복약 리마인더
- 📋 의사 추천 및 소견서
- 📊 가중치 증상 점수
- 📱 모바일 설치 가능 PWA
- 🔒 CSRF 보호 및 속도 제한
- 🌐 다국어 번역

## 🚀 Quick Start

```bash
git clone https://github.com/ImanIP89/MedInNowhere.git
cd MedInNowhere
pip install -r requirements.txt
python database.py
python app.py

Then open:

http://127.0.0.1:5000 (local)

http://hostname.local:5000 (stable PWA connection)

🔑 Default Accounts
Admin: admin / admin123

Doctor: doctor / doctor123
```
